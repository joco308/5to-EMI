# GIT
Es un sistema de control de verciones el cual sirve para gestionar las verciones de un proyecto.  
Tambien para trabajar de forma remota y con colaboradores.  
Para guardar los cambios o poder utilizar esto de forma temota necesitamos un repositorio en la nube.  
- GitLab
- GitHub
- Bitbucket

Solo funciona con archivos de texto plano.
